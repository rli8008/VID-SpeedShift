# VID SpeedShift

VID SpeedShift is a lightweight Firefox extension that lets you control
HTML5 video playback speed on any website.

## Features
### Control video playback speed on any website.
- Slider and numeric speed control
- Preset speed buttons
- Speed memory per tab
- No tracking, no ads

## Description

VID SpeedShift is a clean, lightweight Firefox extension that lets you adjust video playback speed with precision.
Use preset buttons, a slider, or manual input to set speeds up to 7×.
The extension remembers the current speed per tab and respects system dark mode automatically.
No tracking. No ads. No clutter.

## Max Speed
Up to 7× playback speed.
